WebMode transpiles modern JavaScript with Babel and loads styles from Sass/PostCSS. It's opinionated yet minimalist, adding no frameworks or libraries. Mode also copies static assets such as images and documents with a dedicated /static directory.

How to use

npm start to run in dev mode.
npm run build for a production build into a /dist directory, with a main.css stylesheet.
